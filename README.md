### Prerequisites

Make sure you have the required tools installed before running this project.

### Installation

```bash
# Clone the repository
git clone https://github.com/salman-dev-app/FileToLink.git
cd FileToLink

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp sample.env .env
# Edit .env with your API_ID, API_HASH, BOT_TOKEN

# Start the bot
python main.py
```
