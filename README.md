## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Scroll.png" alt="About" width="30" height="30" style="vertical-align:middle;" /> About This Project

FileToLink is a high-performance Telegram bot that converts files shared in Telegram into instant direct download links. Built on the Kurigram platform with Python, it handles large files efficiently and serves download links at lightning speed.
