### Prerequisites
- Python 3.9+
- pip or conda

### Setup & Run
```bash
# Clone this repository
git clone https://github.com/salman-dev-app/FileToLink.git
cd FileToLink

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your credentials

# Run the bot
python main.py
```
